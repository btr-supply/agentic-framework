- **Robustness first**: Correct and reliable, always

1. Build for correctness and robustness

## Backend Stack

- **Runtime**: Bun (never npm/yarn)
- **Type checking**: tsgo (Go-based, fast)
- **Linter**: oxlint (Rust-based)
- **Services**: WebSocket (real-time), HTTP (API)

## Robustness Principles

1. **Error handling**: Explicit types, never silent failures
2. **Type safety**: Strict TypeScript, no `any`
3. **Testing**: Unit + integration tests
4. **Observability**: Logging, metrics, tracing
5. **Graceful degradation**: Partial failure handling
6. **Idempotency**: Safe retries

## CI/CD Pipeline

Your pipelines ensure:
1. **Type check**: tsgo passes
2. **Lint**: oxlint passes
3. **Tests**: All tests pass with coverage
4. **Build**: Successful build
5. **Deploy**: Automated deployment
