# Growth & Marketing Workflow

**Orchestrator**: Sybil → Draper (Growth Lead)
**Support**: Edna (Brand), Jocasta (Content), Seldon (Data-backed claims)

---

## Purpose

Marketing campaigns, social media management, post drafting, community building, and growth initiatives.

---

## When to Use This Workflow

| Trigger | Description |
|---------|-------------|
| New feature launch | Announce to community, educate users |
| Campaign | Planned marketing initiative |
| Content calendar | Regular social media, blog posts |
| Community event | AMA, trading competition, liquidity incentives |
| Partnership announcement | Cross-promotion with other protocols |

---

## Workflow Steps

### Step 1: Campaign Planning (Draper)

```
Draper:
1. WEBSEARCH: Latest DeFi marketing strategies, crypto social media trends
2. Define objective: Onboarding users? Increasing TVL? Brand awareness?
3. Identify target audience: Retail? Institutional? Both?
4. Set KPIs: Followers, visits, TVL, trading volume
5. Plan content calendar
→ Output: Campaign brief with timeline
```

### Step 2: Brand Alignment (Edna)

```
Edna:
1. Ensure visual assets are professional and on-brand
2. Create illustrations, icons, graphics as needed
3. Review all content for visual consistency
→ Output: Brand assets
```

### Step 3: Content Creation (Jocasta + Draper)

```
Jocasta (long-form) + Draper (short-form):

LONG-FORM (Jocasta):
1. Research-backed articles
2. Technical deep-dives
3. Integration guides
4. Publish on Medium, LinkedIn Pulse

SHORT-FORM (Draper simplifies Jocasta's content):
1. Twitter/X threads (daily/weekly)
2. Discord announcements
3. Telegram updates
4. Social media copy
```

### Step 4: Data-Backed Claims (Seldon)

```
For any quantitative claims:
Seldon:
1. Verify data: Historical APY, volume, efficiency metrics
2. Provide evidence: Screenshots, on-chain data
3. Ensure accuracy: Never misrepresent capabilities
→ Output: Verified data points for marketing
```

### Step 5: Campaign Execution (Draper)

```
Draper:
1. Execute content calendar
2. Engage with community: Discord, Telegram, Twitter
3. Coordinate with KOLs/influencers if applicable
4. Track engagement and metrics
→ Output: Active campaign with metrics
```

### Step 6: Review & Optimize (Sybil + Draper)

```
Sybil reviews:
1. Is messaging accurate? (No misrepresentation)
2. Is it consistent with brand voice?
3. Are KPIs being met?
4. What should we adjust?

Draper optimizes based on data
→ Output: Campaign performance report
```

## Key Messages

1. **Single-sided LP**: Deposit one token, receive fungible LP tokens
2. **Capital efficiency**: Higher returns per dollar
3. **Multi-asset**: Trade any-to-any through anchor tree
4. **Cooperative arbitrage**: Profits return to LPs
5. **Security first**: Audited, battle-tested

---

## Messaging Do's and Don'ts

| Do | Don't |
|-----|-------|
| "Earn passive income" | "Guaranteed returns" |
| "Historical APY: 15%" | "You will earn 15%" |
| "Advanced risk management" | "Risk-free" |
| "Competitive fees" | "Lowest fees" (without proof) |
